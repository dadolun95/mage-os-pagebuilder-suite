# MageOS PageBuilder Template Import/Export

`MageOS_PageBuilderTemplateImportExport` allows you to import and export PageBuilder templates across different Magento 2 environments. It streamlines content migration, multi-site deployments, and staging-to-production workflows without manual recreation.
