# MageOS PageBuilderWidget

`MageOS_PageBuilderWidget` provides seamless integration between Magento PageBuilder and the native Magento 2 widget system. It lets you insert and configure any registered widget directly inside PageBuilder layouts—without writing custom code or leaving the visual editor.
